# 🧮 Digital Calculator

A simple **Digital Calculator** built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations through an interactive calculator interface.

## ✨ Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🔢 Decimal number support
* 🎨 Attractive user interface
* 📱 Responsive button layout
* ⚡ Fast and simple calculations

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

```text
Digital-Calculator/
│── index.html
└── README.md
```

## 🚀 How to Run

1. Clone this repository.
2. Open the project folder.
3. Open `index.html` in any web browser.

No additional software or installation is required.

## 🔮 Future Improvements
